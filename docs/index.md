# Welcome to AI Workbench Documentation

## Currently this project is Tomb-stoned [Depricated]

Welcome to the official documentation for the **AI-workbench**. This project provides a collection of AI agents designed to handle various tasks, from simple stateless queries to complex planning and summarization.

### [Repository Link Click Here](https://github.com/thecodeshastra/AI-workbench)

## Core Features

- **Stateless Agent**: A lightweight agent for one-shot request/response interactions. Ideal for quick queries and code generation.
- **Summarizer Agent**: An advanced agent capable of planning and executing tasks, specifically designed for summarizing content and file operations with async support.
- **Virtual Assistant Agent**: A persona-based chatbot that represents a real person, learns from interactions, and manages a personality via PDF/text context.
- **Extensible Architecture**: Built on a modular provider interface, allowing easy integration with different LLM backends.
